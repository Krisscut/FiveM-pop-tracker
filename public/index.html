About

This website has been created to help a community to check its progression population wise relatively to the others Five M Servers

It uses Node JS as backend server, SemanticUI as UI framework and (soon) will use AngularJS 2.0 as frontend framework

Please report any bug that you encounters on github! (check the links in the footer)
